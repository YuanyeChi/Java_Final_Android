# 英语刷题神器
> 这是一款面向需要加强高考英语能力的考生的刷题app,包含**词汇,句式,阅读,完型,作文等1155道高考真题**以及近几年的模拟题.

> 用户可以通过半自主选题,智能选题,以及自主选题(建议搭配错题本使用)三种方式进行刷题,并提供动态生成的结果报告.

> 同时app还带有联网登录注册功能,并有一个简易教师端可以对多位学生的作文进行批改.
